# 프론트엔드 (React + TypeScript + Vite)

RAG Chatbot의 React 프론트엔드 애플리케이션입니다.

## 설치

```bash
cd frontend
npm install
```

## 환경 변수 설정

`.env.example` 파일을 참고하여 `.env` 파일을 생성하고 필요한 환경 변수를 설정하세요.

기본값:
- `VITE_API_URL=http://localhost:8000`

## 실행

```bash
# 개발 모드
npm run dev

# 빌드
npm run build

# 빌드 미리보기
npm run preview
```

개발 서버는 기본적으로 `http://localhost:5173`에서 실행됩니다.

## 기술 스택

- React 18
- TypeScript
- Vite
- TailwindCSS
